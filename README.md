## EKA Classifier

A websocket relay server written in NodeJS.

Part on the [Estonian Academy of Arts](https://www.artun.ee/en/home/) Emerging Themes masters project.

### Demo

https://eka-server.now.sh

### Installation and running locally

```sh
npm i
npm run start
```

### Deploying

```sh
npm i -g now
now
now alias
```
